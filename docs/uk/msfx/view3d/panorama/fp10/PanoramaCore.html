The core class of the Panorama Framework that handles the initialisation of the 3D scene in which a panoramic image is mapped to the inside of a sphere.

This class requires Away3D 3.6.0.

PanoramaCore()Constructor
public function PanoramaCore(image:*, radius:int = 200, segmentsX:int = 20, segmentsY:int = 20, viewWidth:int = 1024, viewHeight:int = 768, stats:Boolean = false)

Constructor for the Panorama Core.

If all you need is a Sphere with a panoramic image without any interactivity, i.e. it rotates by itself then this class will suffice.

Parameters
image:* — The panoramic image to be used, in either BitmapMaterial or MovieMaterial format
 
radius:int (default = 200) — OPTIONAL The radius of the sphere we're rendering onto (adjustable at runtime)
 
segmentsX:int (default = 20) — OPTIONAL The number of segments to make up the sphere (x axis)
 
segmentsY:int (default = 20) — OPTIONAL The number of segments to make up the sphere (y axis)
 
viewWidth:int (default = 1024) — OPTIONAL The width of the 3d view (resizable at runtime via ".width")
 
viewHeight:int (default = 768) — OPTIONAL The height of the 3d view (resizable at runtime via ".height")
 
stats:Boolean (default = false) — OPTIONAL Whether to include Away3D stats on RMB
