Classpublic class InteractivePanorama
InheritanceInteractivePanorama PanoramaCore flash.display.Sprite

Adds the ability to navigate around the Panorama with the Mouse to the PanoramaCore class.

Also adds the MouseWheel Zoom functionality

This class requires Away3D 3.6.0.

InteractivePanorama()Constructor
public function InteractivePanorama(image:*, radius:int = 100, segmentsX:int = 10, segmentsY:int = 10, viewWidth:int = 1024, viewHeight:int = 768, stats:Boolean = false)

Constructor for the Interactive Panorama Class.

If you wish to interact with the panorama, i.e. Click and Glide / Mouse Zoom then use this Class.

Parameters
image:* — The panoramic image to be used, in either BitmapMaterial or MovieMaterial format
 
radius:int (default = 100) — OPTIONAL The radius of the sphere we're rendering onto (adjustable at runtime)
 
segmentsX:int (default = 10) — OPTIONAL The number of segments to make up the sphere (x axis)
 
segmentsY:int (default = 10) — OPTIONAL The number of segments to make up the sphere (y axis)
 
viewWidth:int (default = 1024) — OPTIONAL The width of the 3d view (resizable at runtime via ".width")
 
viewHeight:int (default = 768) — OPTIONAL The height of the 3d view (resizable at runtime via ".height")
 
stats:Boolean (default = false) — OPTIONAL Whether to include Away3D stats on RMB
